# Study-HSK3-new-word
<p>An API to learn HSK3 vocabulary by typing pinyin
  </p>
<p>-- run python app.py to run the app
  </p>
<p>-- turn on Chinese typing mode on the computer, then look at the word on the screen and type the corresponding pinyin
  </p>
